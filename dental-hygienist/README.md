# Dental Hygienist Plugin for Claude

Document periodontal findings, write insurance narratives, and create patient education materials.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/dental-hygienist) — AI tools, guides, and weekly digests designed specifically for dental hygienists.

## Commands

| Command | Description |
|---------|-------------|
| `/perio-narrative` | Draft periodontal clinical narratives from probing data, bleeding sites, and radiographic findings using AAP classification |
| `/insurance-narrative` | Write clinical justification narratives for dental insurance claims including periodontal treatment, radiograph frequency, and SRP |
| `/patient-education` | Generate patient education materials including post-procedure instructions, oral hygiene education, and condition-specific handouts |
| `/treatment-summary` | Create treatment plan summaries from clinical findings with urgency classification, sequencing, and patient consent talking points |

## Skills

This plugin includes skills that activate automatically when you're working on dental hygiene-related tasks:

- **Clinical Dental Hygiene** — Periodontal assessment, AAP classification, ADA CDT coding, radiograph interpretation, instrument selection, and dental pharmacology
- **Oral Health Education** — Motivational interviewing for behavior change, age-appropriate patient education, condition-specific messaging, and culturally sensitive communication

## Usage examples

```
/perio-narrative
Patient presents for perio re-evaluation. Probing depths: #3 DL 6mm, #14 ML 5mm,
#19 DL 7mm, #30 ML 5mm. Generalized 4mm pockets posteriors. BOP 45%.
Radiographs show moderate horizontal bone loss posteriors, vertical defect
#19 distal. Patient is Type 2 diabetic, HbA1c 7.8, former smoker quit 2 years ago.
Previous SRP completed 3 months ago.
```

```
/insurance-narrative
Need clinical narrative for SRP D4341 all four quadrants. Patient has generalized
5-6mm pockets, 55% BOP, radiographic bone loss moderate-severe posteriors.
AAP Stage III Grade B generalized. Patient is 52yo, Type 2 diabetic on metformin.
Previous prophy 6 months ago with no improvement. Insurer: Delta Dental.
```

```
/patient-education
Post-SRP instructions for a patient who just had lower right quadrant scaling
and root planing under local anesthesia. Patient is anxious about aftercare.
Low health literacy. Include what to eat, pain management, and when to call.
Also include a section on why the follow-up appointment in 4-6 weeks matters.
```

```
/treatment-summary
Findings from today's exam: #3 MOD composite failing with recurrent decay,
#14 mesial caries (moderate), #19 needs crown (cracked tooth with large existing
restoration), generalized Stage II periodontitis needing SRP all quads, and
overdue for BWX. Patient has Delta Dental PPO with $1500 annual max, $800 remaining.
Organize by urgency and help me present this to the patient.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute dental or medical advice. The dental hygienist and supervising dentist are responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
