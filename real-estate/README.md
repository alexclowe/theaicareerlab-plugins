# Real Estate Agent Plugin for Claude

Write listing descriptions, draft client emails, and create market analysis narratives.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/real-estate) — AI tools, guides, and weekly digests designed specifically for real estate agents.

## Commands

| Command | Description |
|---------|-------------|
| `/listing-description` | Generate MLS, social media, and marketing descriptions from property details |
| `/client-email` | Draft client communication emails for real estate scenarios |
| `/cma-narrative` | Turn comparable sales data into persuasive market analysis narratives for sellers |
| `/market-update` | Generate neighborhood or area market update emails for your sphere of influence |
| `/offer-comparison` | Help sellers navigate multiple offers with side-by-side comparison and analysis |

## Skills

This plugin includes skills that activate automatically when you're working on real estate tasks:

- **Real Estate Marketing** — MLS listing writing, social media marketing, Fair Housing Act compliance, property photography direction, and open house marketing
- **Client Communication** — Real estate client relationship communication for buyers, sellers, and leads including price reductions, contingency explanations, escrow updates, and cold lead re-engagement

## Usage examples

```
/listing-description
4BR/3BA, 2400 sqft colonial in Westfield, NJ. Updated kitchen with quartz
counters and stainless appliances. Hardwood floors throughout main level.
Finished basement with wet bar. Fenced backyard with paver patio. Built
2005, new roof 2022. Asking $725,000.
```

```
/client-email
Price reduction conversation with seller "Karen Mitchell." Home at 45 Oak
Street listed at $599K for 42 days, only 6 showings. Feedback: price too
high for the updates. Comps suggest $549-565K range. Recommend $565K.
```

```
/cma-narrative
Subject: 3BR/2BA ranch, 1800 sqft, updated kitchen, original baths, corner
lot, Maplewood NJ. Comps: 1) 22 Elm sold $485K, 1750 sqft, full reno,
45 days ago. 2) 18 Oak sold $462K, 1900 sqft, original condition, 30 days
ago. 3) 7 Pine sold $478K, 1820 sqft, updated kitchen+baths, 60 days ago.
```

```
/market-update
Area: Downtown Austin, TX. Q4 2025. Median price: $485K (up 3% YoY).
Inventory: 2.1 months. Avg DOM: 28 days (down from 35). List-to-sale:
97.2%. New construction starts up 15%.
```

```
/offer-comparison
Listing: 123 Main St, asking $550K. Offer 1: $560K, conventional 20% down,
no contingencies, close in 30 days. Offer 2: $575K with escalation to $590K,
FHA 3.5% down, inspection + appraisal contingency, close in 45 days.
Offer 3: $550K cash, as-is, close in 14 days, rent-back 30 days.
```
