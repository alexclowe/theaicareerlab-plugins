# Bookkeeper Plugin for Claude

Draft financial summaries, prepare reconciliation reports, and generate client communications.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/bookkeeper) — AI tools, guides, and weekly digests designed specifically for bookkeepers.

## Commands

| Command | Description |
|---------|-------------|
| `/financial-summary` | Generate narrative financial summaries that business owners can actually understand |
| `/reconciliation-report` | Draft monthly bank reconciliation reports with clear documentation of reconciling items |
| `/tax-prep` | Generate tax preparation documentation, checklists, and client communications by entity type |
| `/client-email` | Create professional bookkeeping client communications for updates, reminders, and onboarding |

## Skills

This plugin includes skills that activate automatically when you're working on bookkeeping tasks:

- **Financial Reporting** — Financial statement preparation, analysis, and plain-language explanation for small business owners, including accrual vs cash basis, adjusting entries, and GAAP references
- **Tax Communication** — Tax preparation client communication, deadline management, entity-specific requirements, and common deduction guidance

## Usage examples

```
/financial-summary
Client: "Bright Side Coffee" (single-location cafe). March 2026.
Revenue: $47,200 (up from $42,800 in Feb). COGS: $18,100. Payroll: $14,500.
Rent: $4,200. Utilities: $890. Marketing: $1,200. Insurance: $450.
Other expenses: $2,100. Notable: $3,800 equipment repair (espresso machine).
Cash on hand: $12,400. AR: $0. AP: $3,200 due within 30 days.
```

```
/reconciliation-report
Business checking — period ending Feb 28, 2026. Bank statement balance:
$34,567.89. Book balance: $33,245.67. Outstanding checks: #1842 ($450),
#1847 ($312.22), #1850 ($890). Deposit in transit: $1,100 (deposited 2/28).
Bank charges: $45 monthly fee, $12 wire fee. Interest earned: $2.80.
Unidentified charge: $75 on 2/15 — need client confirmation.
```

```
/tax-prep
Entity: S-Corp. Tax year: 2025. Client: "Martinez Design Studio LLC."
2 shareholders (50/50). Have: P&L, balance sheet, bank statements, payroll
records (W-2s issued). Missing: 1099s received, vehicle mileage logs,
home office details for one shareholder, Q4 estimated payment confirmation.
Filing deadline: March 15, 2026.
```

```
/client-email
Tax deadline reminder for "Rebecca Torres" — sole proprietor, freelance
graphic designer. Q1 estimated payment due April 15. Last year she owed
$8,400 total, paid quarterly. Also need her Q1 income summary to calculate
this quarter's estimate. She tends to submit documents late.
```
