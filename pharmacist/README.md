# Pharmacist Plugin for Claude

Review drug interactions, check dosing protocols, and draft patient counseling notes.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/pharmacist) — AI tools, guides, and weekly digests designed specifically for pharmacists.

## Commands

| Command | Description |
|---------|-------------|
| `/drug-interaction` | Check drug interactions for a medication list with severity ratings and management recommendations |
| `/patient-counsel` | Draft patient counseling scripts tailored to medication, literacy level, and counseling context |
| `/prior-auth` | Generate evidence-based prior authorization letters for non-formulary or denied medications |
| `/mtm-note` | Draft MTM notes and comprehensive medication review summaries |

## Skills

This plugin includes skills that activate automatically when you're working on pharmacy-related tasks:

- **Clinical Pharmacy** — Drug information, pharmacotherapy, dosing protocols, formulary navigation, and regulatory awareness
- **Patient Communication** — Health literacy-adapted counseling scripts, patient handouts, and teach-back frameworks

## Usage examples

```
/drug-interaction
Evaluate interactions for: metformin 1000mg BID, lisinopril 20mg daily,
atorvastatin 40mg daily, and amlodipine 10mg daily. Patient is 68yo with
CrCl 45 mL/min.
```

```
/patient-counsel
New start counseling for warfarin 5mg daily. Patient is 72yo, low health
literacy, starting after DVT diagnosis. Include what to avoid and when to
seek emergency care.
```

```
/prior-auth
Drug: Eliquis 5mg BID. Indication: DVT prophylaxis post-TKR.
Patient failed warfarin (labile INR, 2 ER visits for supratherapeutic INR).
Insurer: Aetna. Urgency: Urgent — patient currently without anticoagulation.
```

```
/mtm-note
CMR for patient on 12 medications. Findings: duplicate calcium channel blocker
therapy (amlodipine + diltiazem), no statin despite diabetes + CAD, metformin
dose not adjusted for CrCl 38. Recommended discontinuing diltiazem, adding
atorvastatin, reducing metformin to 500mg BID.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute medical or pharmaceutical advice. The pharmacist is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
