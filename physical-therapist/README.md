# Physical Therapist Plugin for Claude

Draft SOAP notes, build exercise programs, and write insurance appeal letters.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/physical-therapist) — AI tools, guides, and weekly digests designed specifically for physical therapists.

## Commands

| Command | Description |
|---------|-------------|
| `/soap-note` | Draft complete SOAP notes from session observations for initial evaluations, daily treatments, and re-evaluations |
| `/insurance-appeal` | Generate evidence-based medical necessity letters for denied physical therapy visits |
| `/exercise-program` | Build Home Exercise Programs with clear patient instructions, progressions, and printable formatting |
| `/progress-note` | Draft progress notes documenting functional improvements, goal status, and continued treatment justification |

## Skills

This plugin includes skills that activate automatically when you're working on physical therapy-related tasks:

- **Rehabilitation Medicine** — Evidence-based PT interventions, clinical practice guidelines, functional outcome measures, ICD-10 coding, and insurance documentation requirements
- **Patient Exercise Instruction** — Writing clear, safe exercise instructions adapted to patient literacy levels with proper safety cues and progression guidance

## Usage examples

```
/soap-note
Initial evaluation for 55yo female, R shoulder pain x 6 weeks after fall.
AROM: flexion 95 degrees, abduction 80 degrees, IR 30 degrees. MMT: supraspinatus
3/5, deltoid 4/5. Positive Neer's and Hawkins-Kennedy. DASH score 68.
Dx: R rotator cuff tendinopathy. Plan: 2x/week x 8 weeks.
```

```
/insurance-appeal
Denial for continued PT visits. Patient: 72yo male, L TKR 4 weeks ago.
Currently at 85 degrees flexion (need 120 for functional independence),
ambulating with rolling walker, mod assist for stairs. LEFS improved
from 12 to 28 (MCID is 9). Insurer: UnitedHealthcare. Need 12 more visits.
```

```
/exercise-program
HEP for patient with chronic low back pain, currently tolerating core
stabilization exercises. No radiculopathy. Goals: return to recreational
golf. Include: lumbar stabilization, hip strengthening, thoracic mobility.
Patient is 60yo, active, good health literacy. Precaution: avoid loaded
flexion.
```

```
/progress-note
Progress note for 45yo female, cervicogenic headache and neck pain.
Treated 8 of 12 authorized visits. NDI improved from 52% to 28%.
Cervical AROM improved 40%. Headache frequency decreased from daily to
2x/week. Still has limited upper cervical mobility and postural deficits.
Need 6 more visits to meet discharge goals.
```

## Disclaimer

All clinical content generated by this plugin is for informational and drafting purposes only. It does not constitute medical or therapeutic advice. The physical therapist is responsible for verifying all clinical information and exercising independent professional judgment before clinical use.
