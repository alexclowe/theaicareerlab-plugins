# Management Consultant Plugin for Claude

Draft client proposals, write executive summaries, and turn meetings into action plans.

Built by [The AI Career Lab](https://theaicareerlab.com/professions/management-consultant) — AI tools, guides, and weekly digests designed specifically for management consultants.

## Commands

| Command | Description |
|---------|-------------|
| `/proposal` | Generate structured consulting proposals for client engagements with phased work plans and fee structures |
| `/executive-summary` | Distill complex project findings into clear, audience-adapted executive summaries |
| `/strategy-memo` | Create structured strategy memos using pyramid principle and consulting frameworks |
| `/meeting-followup` | Transform meeting notes into structured summaries, action items, and professional follow-up emails |

## Skills

This plugin includes skills that activate automatically when you're working on consulting-related tasks:

- **Strategic Analysis** — Management consulting frameworks (Porter's Five Forces, BCG matrix, McKinsey 7S), MECE problem structuring, hypothesis-driven analysis, pyramid principle communication, and financial analysis
- **Client Deliverables** — Creating polished client-facing documents with clear visual hierarchy, quantified impact, action titles, and top-tier consulting presentation norms

## Usage examples

```
/proposal
Client: Meridian Health Systems. Problem: Post-merger integration of two regional
hospital networks — duplicated administrative functions, misaligned IT systems,
and physician dissatisfaction. Engagement type: Operations improvement.
Timeline: 16 weeks. Fee: Fixed fee, $450K across 4 phases.
```

```
/executive-summary
Audience: C-suite. Project: 6-week cost optimization diagnostic for a mid-market
manufacturing company. Key findings: 23% overhead cost premium vs industry
benchmark, $8.2M in addressable spend across procurement and facilities,
quick wins available in vendor consolidation. Recommendations: consolidate
top 3 vendor categories, renegotiate facilities contracts, implement demand
management for indirect spend.
```

```
/strategy-memo
Topic: Market entry strategy for Southeast Asia expansion. Background: Company
currently operates in 4 European markets with $120M revenue. Board has approved
exploration of ASEAN markets. Analysis: Vietnam and Indonesia offer strongest
fit based on market size, competitive intensity, and regulatory environment.
Recommend phased entry starting with Vietnam via local partnership model.
```

```
/meeting-followup
Attendees: Sarah Chen (VP Strategy), Marcus Johnson (CFO), Lisa Park (Head of Ops),
David Kim (Project Lead). Meeting: Weekly steering committee for cost transformation
program. Notes: Sarah opened with update on vendor negotiations — Acme contract
renegotiated, saving $1.2M annually. Marcus flagged Q3 budget pressure, needs
interim savings report by Friday. Lisa raised concern about warehouse staffing
cuts affecting fulfillment SLAs. David to present revised timeline next week.
Agreed to accelerate Phase 2 by two weeks.
```

## Disclaimer

This output is a professional draft — review and customize before client delivery. The consultant is responsible for verifying all analysis, tailoring recommendations to specific client contexts, and exercising independent professional judgment.
